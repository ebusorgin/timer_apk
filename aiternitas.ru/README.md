# Aiternitas

Главная страница и лендинг платформы Aiternitas.

## Описание

Простая главная страница с боковым меню проектов и информацией о платформе.

## Установка

```bash
npm install
```

## Запуск

```bash
npm start
```

Сервер будет доступен на `http://localhost:3001`

## Деплой

Проект автоматически деплоится на `https://aiternitas.ru` при push в ветку `production`.

## Структура проекта

- `server.mjs` - Express сервер
- `index.html` - Главная страница (лендинг)

## Технологии

- Node.js
- Express
