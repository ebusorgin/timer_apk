ИНСТРУКЦИЯ: Добавление SSH ключа в GitHub
===========================================

ШАГ 1: Откройте страницу Secrets
https://github.com/ebusorgin/timer_apk/settings/secrets/actions

ШАГ 2: Нажмите "New repository secret"

ШАГ 3: Заполните форму:
  Name: SSH_PRIVATE_KEY
  Secret: (скопируйте ключ ниже, начиная с -----BEGIN)

ШАГ 4: Нажмите "Add secret"
